# /voiceover — Voiceover Director

Adds natural voiceover narration to an existing video generated by `/video`.

## Usage

```
/voiceover <VideoName>
```

## Examples

```
/voiceover HowTheWebWorks
/voiceover TheAIStackExplained
/voiceover UnderstandingDatabases
```

## What happens

1. **Manifest check** — Reads or generates `manifest.json` for the video
2. **Transcript generation** — Creates a transcript skeleton, then AI writes natural narration for each scene
3. **User review** — You review and edit the transcript before synthesis
4. **TTS synthesis** — Calls Fish Audio (or configured provider) to generate MP3 for each scene
5. **Audio integration** — Adds `VoiceoverLayer` to the video composition

## Prerequisites

- The video must already exist in `src/<VideoName>/` (generated by `/video`)
- `FISH_AUDIO_API_KEY` must be set in `.env`
- `FISH_AUDIO_VOICE_ID` must be set in `.env` (clone your voice at https://fish.audio first)

## Voice cloning

1. Record 15-30 seconds of clear audio
2. Go to https://fish.audio and create a voice model
3. Upload your sample, test it
4. Copy the model ID and add to `.env`:
   ```
   FISH_AUDIO_VOICE_ID=<model-id>
   ```

## Environment variables

```bash
TTS_PROVIDER=fish-audio          # default
FISH_AUDIO_API_KEY=<key>         # required
FISH_AUDIO_VOICE_ID=<model-id>  # required — your cloned voice
FISH_AUDIO_MODEL=s1              # s1 | speech-1.6 | speech-1.5
FISH_AUDIO_SPEED=1.0             # 0.5 to 2.0
```
