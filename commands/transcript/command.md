# /transcript — Generate Voiceover Transcript

Generates a narration transcript from an existing video's manifest.

## Usage

```
/transcript <VideoName>
```

## Examples

```
/transcript HowTheWebWorks
/transcript TheAIStackExplained
/transcript UnderstandingDatabases
```

## What happens

1. **Manifest check** — Reads `manifest.json` (with `narrationIntent` from `/video`)
2. **Transcript skeleton** — Generates skeleton via `generate-transcript.ts` with empty narration fields
3. **AI narration** — Writes natural narration for each scene, guided by `narrationIntent`, `onScreenText`, and `narratorTone`
4. **User review** — Presents the complete transcript for editing before proceeding

## Prerequisites

- The video must already exist in `src/<VideoName>/` (generated by `/video`)
- `manifest.json` must exist with scene data

## After review

Run `/voiceover <VideoName>` to synthesize TTS audio and integrate into the composition.
